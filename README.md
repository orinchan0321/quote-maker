# 見積書制作ツール
このツールは、複数ユーザーで利用可能な見積書作成支援アプリケーションです。顧客管理・商品管理・PDF出力・履歴管理など、実務に即した機能を備えています。

## ユーザー管理
- ユーザー名＋パスワードによるログイン
- GoogleアカウントによるOAuthログイン
- 複数ユーザー対応（チーム利用を想定）

## 顧客管理機能
- 顧客名、住所、担当者、登録番号、連絡先の登録・検索
- 顧客詳細の編集・削除
- 顧客情報の再利用

## 商品・サービス管理
- 商品名、単価、数量、税率の登録
- 小計・合計の自動計算
- 商品一覧の編集・削除

## 見積書作成機能
- 入力フォームから商品を選択・数量入力
- 顧客選択による情報自動入力
- 消費税・割引・合計金額の自動算出
- 見積書プレビュー表示
- 
## 出力・共有機能
- PDF生成（jsPDFまたはPuppeteerを使用）
- ダウンロードまたはメール送信
- クラウド共有（URL発行による外部共有）

## 履歴・ステータス管理
- 過去の見積書の検索・再利用
- ステータス管理（作成中／送付済み／承認済み）
- ローカルストレージ（AES暗号化）による保存

## 差別化機能
- 見積書 → 請求書・納品書への自動変換（インボイス対応）
- テンプレート切り替え（複数デザインから選択可能）
- 顧客とのクラウド共有機能
